## Cara Bermain
- Gerakkan bom ke arah pocong dan ledakkan
- Gunakan tombol panah kiri, kanan, atas, bawah untuk menggerakkan bom
- Gunakan tombol spasi untuk meledakkan bom
- Gunakan tombol R untuk mereset game
